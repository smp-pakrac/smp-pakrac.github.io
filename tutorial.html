RESOURCE PACK

natrag

ovo napravi prije preuzimanja modova

  1. preuzmi resource pack
  2. izreži file (CTRL + X)
  3. mazni windows + r
  4. ukucaj %appdata%
  5. uđi u .minecraft folder
  6. uđi u resourcepacks folder
  7. zaljepi file (CTRL + V)

PREUZIMANJE

  1. preuzmi temeljne modove
  2. otvori file
  3. desni klik na prostor bez fajlova
  4. klikni na "izdvoji sve"
  5. u novom prozoru klikni pregledaj
  6. klikni na prostor istaknut crvenom bojom i tamo upiši %appdata% zatim mazni enter
  7. uđi u .minecraft folder
  8. uđi u mods folder
  9. stisni odaberi
  10. stisni izdvoji
